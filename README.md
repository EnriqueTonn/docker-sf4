# Docker SF4

> Simple example of Symfony 4 with Docker (docker-compose)



### Installing

Clone this repository

```sh
$ git clone https://github.com/migueabellan/docker-sf4.git
```

Install with composer

```sh
$ composer install
```

Test

```sh
$ phpunit tests
```

Up and down docker

```sh
$ docker-compose up -d
localhost:8000

$ docker-compose down
```
